## Неизменные шаблонные блоки

В финальный README вставь следующие блоки дословно. Не меняй заголовки, порядок, формулировки,
ссылочный смысл и пунктуацию внутри блоков.

```markdown
{{blocks_markdown}}
```

Если направление переопределяет блоки в профиле, используй профильные тексты как источник истины.
