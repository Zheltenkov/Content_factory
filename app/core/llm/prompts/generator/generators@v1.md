Сгенерируй только дополнительные подсказки для тяжелых генераторов G4c.

Верни JSON, совместимый со схемой `GeneratorDraft`:

```json
{
  "code_examples": [
    {
      "part_index": 1,
      "label": "краткое название",
      "language": "python",
      "code": "рабочий пример до 30 строк",
      "explanation": "1-3 предложения"
    }
  ],
  "dataset_notes": {
    "materials/source.csv": "какие сырые строки и шум добавить"
  }
}
```

Ограничения:
- не генерируй формулы, таблицы и Mermaid: они строятся детерминированным кодом;
- dataset_notes не должны содержать готовое решение студента;
- code_examples нужны только для hard-code/low-code проектов и должны быть короткими;
- не добавляй markdown вне JSON.
