Сгенерируй Главу 2 "Теория" для учебного README.

Верни только JSON, совместимый со схемой `TheoryDraft`:

```json
{
  "parts": [
    {
      "title": "краткое название части",
      "body": "основной текст без заголовка",
      "example": "1-3 предложения практического примера",
      "bridge_questions": ["1-2 вопроса, привязанные к проекту"]
    }
  ]
}
```

Требования:
- количество частей держи в диапазоне из `thresholds.parts`;
- каждая часть должна раскрывать отдельную тему текущего проекта, а не следующие проекты блока;
- в каждой части добавь минимум одно определение в формате `**Термин** — это ...`;
- пример должен быть внешним кейсом или практической ситуацией, не инструкцией по сдаче;
- вопросы к практике должны быть связаны с инструментом, артефактом или результатом проекта;
- не добавляй Главу 1, Главу 3, P2P-инструкции, правила сдачи и чек-листы.
